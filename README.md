# Prosthetic Outreach Initiative

> GitHub project for coordinating volunteer tasks across the prosthetic outreach initiative teams in **Chicago, Boston, and Atlanta**.

## 🎯 Mission
Finding and supporting affordable prosthetic solutions for underserved communities through clinical research, community outreach, and technical coordination.
